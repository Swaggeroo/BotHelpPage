Important Commands

>getStats

This command returns the overall stats since our bot startes working. It counts the sent messages, the sent bot-commands and the number of created games

Util Commands

>mypicture

This command replies with your profile picture.

>miesmuschel

This command gives a random answer. Do not write the question behind it, because then it won't work (Use the >rand-command instead)

>rand [question]

This command gives also gives a random answer, but you are required to write the Question behind the command

>help

This command leads you to this fabulous website of bot-commands.

>jetztredeich

This awesome command can only be used by moderators and admins. It mutes everyone who is in the same voice-channel as the command writer for 10 seconds (the command writer does not get muted, so for 10 seconds you can talk without getting interrupted)

Game Commands

>creategame

This command returns an overview over all available minigames with it's specific index.

>creategame [index]

This command creates a new textchannel in which the minigame will take place. The minigame is decided by the provided index. Further instructions on how to play are displayed in the channel.

>inv @[user] && >inv everyone

By default only the game-creator (and the admin) can write in the created game- textchannel. The game-creator has to invite other users so that they can write. He can do this with the >inv-command. >inv @[user] invites one specific user while >inv everyone invites every user on the server.

>:mop:

This command can only be used in a game-textchannel. It clears all previously existing messages in the channel (GetMop(p)ed).

>deletegame

With this command the game-creator (or an moderator) can delete the game- textchannel and delete the all data of the game from our DB. To keep a nice and clean DC-Server and to sop our DB from breaking, we want everyone to use this command to clean up after yourself.

Available Games

>Galgenmännchen

Under >creategame 1 you can find our first minigame "Galgenmännchen". I think everybody knows the game, so I will only be covering the commands. At the beginning the game-creator has to choose a word and send it to the bot with the >ga-command. The game starts as soon as the word is chosen and was successfully received by the bot. The word is compared to our blacklist of curse-words, so don't even try to use one.

The players can guess simply by typing >[letter]. It always counts the first letter that is typed after the prefix >, ignoring the rest of the message. You can guess the whole word with the >word-command.
To start a new game, you have to delete the old textchannel and then create a new one with the >creategame 1-command

Money Commands

The money does not have a use yet!

>getMoney

This command returns the amount of money you have at the moment.

>addMoney [amount]

With this command you can add a specific amount of money to your account.

>removeMoney [amount]

With this command you can reduce your money by a specific amount

>daily

Once a day, you can collect 100 money through this command. It is reset at 0:00 PM UTC (1:00 AM UTC+1)

>enablenotification

With this command you can activate/deactivate the notification for an unclaimed daily reward.
